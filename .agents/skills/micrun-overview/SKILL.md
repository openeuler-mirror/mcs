# MicRun 总览

## 概览

当用户任务明确落在 `micrun/` 子项目时，使用这个 skill 作为入口。

这个 skill 建立在 `../mica-overview/SKILL.md` 已经提供的 MICA 基础背景之上。

这个 skill 直接索引到 `micrun/AGENTS.md`。

## 使用场景

- 用户问题明确针对 `micrun/` 目录

不适用于：
- 纯 MICA 主仓生命周期、通信、pedestal 或 service 问题

## 相关入口

从这里开始：
- `micrun/AGENTS.md`

## 阅读指导

- 先具备 `../mica-overview/SKILL.md` 的基础背景模型
- 再阅读 `micrun/AGENTS.md`
